# openWebAccessibility
A simple JavaScript library written in TypeScript for accessibility that can work right out of the box or be customized. It is meant to work well with *openWebAccessibilityStudio*, which makes it easier to implement accessibility in your site as well as lets you assess your site's accessibility.


## Accessibility  Features
### Font Settings
* Set Font
* Set Font Color
* Set Font Styles (Bold, Italic, Coloring)
* Set Font Size
* Set Line Spacing
* Set Character Spacing

### Animation Settings
* Toggle Animations
* Change Animation Speed
* Turn off certain types of animation

### Screen Reader
* Screen Reader Widget
* Screen Reader Settings
  * Voice
  * Speed
  * Pitch
  * Highlighting Settings

### Dictionary
* Definition
* Thesaurus
* Other resources
* Wikipedia Data
* Info Sources

### Visual Dictionary
Pop-ups to describe/define words

### Magnifier
* Configurable magnifier tool
* Full-screen magnification

### Colors
* Saturation Controls
* Contrast Controls
* Transparency Controls
* White Point Controls
* Remove color effects

### Reader Mode
A mode that simplifies webpage content to make it easily readable.

### Image Settings
* Color settings for images
* Image resizing
* Image Disabling

### Link Settings
* Set font settings for links and buttons
* Highlight links and buttons
* Read links on hover
* Set link colors

### Page Structure
* Page Structure Widget
* Simple way to see and navigate page structure.

### Cursor Settings
* Change Cursor Type
* Change Cursor Size
* Settings for controlling cursor/focus

### Voice Settings
* Voice Toggle Widget
* Speech to Text Options
* Speech Control Page Settings

### Video Setting
* Color settings for videos
* Audio settings for videos
* Video disabling
* Transcription Tools

### Audio Settings
* Speed Controls
* Pitch Controls
* Transcription Tools

### Transcription Tool
* Video and audio navigation tools
* Video/Audio Transcriptions

### Aria Tools
* Simplifying setting up aria tags.

### Language Tools
* Multilingual Dictionary
* Machine Translation Tools

### Easy Integration With *openWebAccessibilityStudio*
* Supports accessibility profiles made with *openWebAccessibilityStudio*
* Enables *openWebAccessibilityStudio* to understand better and integrate your site's accessibility.
